# 📈 AI 주식 투자 비서 (AI Stock Assistant)

**"바이브 코딩(Vibe Coding)만으로 제작된 AI 주식 분석 도구입니다."**

이 프로젝트는 미국 증시와 한국 증시 데이터를 실시간으로 분석하여, 오늘 주목해야 할 종목을 추천해주는 데스크톱 애플리케이션입니다. 복잡한 설치 없이 실행 파일 하나로 누구나 쉽게 사용할 수 있도록 설계되었습니다.

## 🚀 주요 기능

*   **미국/한국 주식 통합 분석**:
    *   **미국장(US)**: 엔비디아, 테슬라, 애플 등 주요 기술주 및 ETF 분석.
    *   **한국장(KR)**: 미국장과의 커플링(동조화) 현상 분석 및 시가총액 상위 종목 스크리닝.
*   **지능형 추천 시스템**:
    *   **뉴스 분석**: 네이버 뉴스 크롤링을 통해 호재(공급계약, 흑자전환 등)와 악재(유상증자, 횡령 등)를 파악.
    *   **차트 분석**: 이동평균선(MA), RSI 등 기술적 지표를 활용한 추세 분석.
    *   **거래량 분석**: 평소 대비 거래량이 폭발하는 종목 포착.
*   **매매 가이드라인 제공**:
    *   단순 추천을 넘어 **매수가, 목표가, 손절가**를 구체적인 가격으로 제시.
*   **사용자 친화적 UI**:
    *   직관적인 탭(Tab) 구조로 국내/미국 주식을 분리하여 표시.
    *   다크 모드(Dark Mode) 적용으로 눈의 피로도 감소.
    *   클릭 한 번으로 상세 분석 사유 확인 가능.

## 🛠 기술 스택 (Tech Stack)

이 프로젝트는 100% **Python**으로 작성되었으며, 다음의 라이브러리와 프레임워크를 사용했습니다.

### Core
*   **Language**: Python 3.x
*   **GUI Framework**: PyQt6 (Modern Desktop UI)
*   **Styling**: pyqtdarktheme (Dark Theme Support)

### Data & Analysis
*   **Data Collection**:
    *   `FinanceDataReader`: 한국 및 미국 주식 데이터 수집 (안정성 확보).
    *   `requests` & `BeautifulSoup`: 네이버 뉴스 실시간 크롤링.
*   **Data Processing**:
    *   `pandas`: 대용량 데이터 프레임 처리 및 시계열 분석.

### Deployment
*   **Build Tool**: PyInstaller (Standalone Executable Creation)

## 📦 설치 및 실행 방법

별도의 Python 설치가 필요 없습니다.

1.  `dist` 폴더 내의 `AI_Stock_Assistant.exe` 파일을 다운로드합니다.
2.  파일을 더블클릭하여 실행합니다.
3.  **[분석 시작]** 버튼을 누르면 AI가 시장을 분석하여 추천 종목을 리포팅합니다.
